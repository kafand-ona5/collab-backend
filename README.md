# collab-backend
A C++ real-time collaboration backend focusing on concurrency, networking, and distributed systems.

## Overview
This project is a C++ backend system that enables multiple users to collaboratively edit shared structured data in real time using WebSockets and CRDTs.

## Why This Project?
Most beginner projects focus on CRUD apps. This project focuses on:
- Concurrency
- Real-time networking
- Distributed systems
- Conflict-free data synchronization
